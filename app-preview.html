성적증명서 추출 구조에 대해

네, 실제 성적증명서 정보를 추출하려면 Upstage API 같은 OCR/Information Extract API가 필요합니다. 이 HTML 미리보기는 브라우저에서 첨부만 보여주고, 실제 파싱 결과는 mock 또는 서버 연동이 있어야 생성됩니다.

원하시면 다음 단계에서 Upstage API 키를 받는 전제로 실제 추출 라우트까지 연결해드릴 수 있습니다.
